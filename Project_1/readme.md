# Instructions to run the program

Make sure to have a GNU complier installed to compile c++ code

1. Unzip and open the folder
2. Run this command to compile

   ```bash
   g++ DiamondPattern.cpp -o run
   ```

3. Run this command to run

   ```bash
   ./run
   ```
